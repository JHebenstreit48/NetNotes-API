# What Is IMAP?

<hr class="dividerSection" />

<span class="emphasis">IMAP</span> stands for <span class="emphasis">Internet Message Access Protocol</span>. A server that supports IMAP clients receives and stores messages addressed to its users, similar to POP3.

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>Unlike POP3, IMAP keeps messages in the mailbox on the server, unless the user deletes them.</li>
    <li>This allows the same mailbox to be accessed consistently from multiple devices, since messages are not downloaded and removed.</li>
    <li>The most current version, <span class="emphasis">IMAP4</span>, listens for client requests on <span class="emphasis">TCP port 143</span>.</li>
  </ul>
</div>

Many different email servers exist for the various network operating system platforms, and most support IMAP alongside SMTP and, in many cases, POP3.
