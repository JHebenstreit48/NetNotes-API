# How Powerline Networking Turns Electrical Wiring Into a Network

<hr class="dividerSection" />

### What Is Powerline Networking?

<hr class="dividerSection" />

<span class="emphasis">Powerline networking</span> adds the ability to connect a device to a network using an adapter plugged into an existing electrical outlet.

The network uses the home's existing electrical wiring to send data.

It is not a replacement for physical cabling, but it can add functionality in places where wireless access points cannot be used or cannot reliably reach devices.

<hr class="dividerSection" />

### How It Works

<hr class="dividerSection" />

A <span class="emphasis">powerline adapter</span> plugs into a standard electrical outlet and connects to a device, such as a router or a computer, using an Ethernet cable.

A second adapter is plugged into an outlet elsewhere in the building, connected to whatever device needs network access at that location.

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>The first adapter sends data over the electrical wiring already installed in the walls.</li>
    <li>The second adapter receives that data and passes it to the connected device over Ethernet.</li>
    <li>Both adapters must generally be on the same electrical circuit or panel to communicate reliably.</li>
  </ul>
</div>

<hr class="dividerSection" />

### When Powerline Networking Is Used

<hr class="dividerSection" />

Powerline adapters are typically used to extend an existing home LAN, rather than to build one from scratch.

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>Reaching a room where a wireless signal is weak or blocked.</li>
    <li>Connecting a device in a location where running new physical cabling is impractical.</li>
    <li>Providing a more stable connection than Wi-Fi for a single device, without rewiring the home.</li>
  </ul>
</div>

<hr class="dividerSection" />

### Limitations

<hr class="dividerSection" />

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>Performance can vary depending on the age and quality of the home's electrical wiring.</li>
    <li>Adapters on different circuits or breaker panels may not communicate reliably.</li>
    <li>Other devices drawing power from the same circuit can introduce electrical noise that affects performance.</li>
    <li>Throughput is generally less consistent than a direct Ethernet connection.</li>
  </ul>
</div>

<hr class="dividerSection" />

### Summary

<hr class="dividerSection" />

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>A device connects to an existing home LAN using an adapter and an existing electrical outlet.</li>
    <li>Powerline networking extends a LAN, it does not replace physical cabling or wireless access points.</li>
    <li>It is most useful where wireless coverage is weak and new cabling is not practical.</li>
  </ul>
</div>
