# T568A vs T568B

---

## Ethernet Wire Color Coding

---

Ethernet twisted-pair cables contain **four color-coded wire pairs**, for a total of **eight wires**.

Each pair consists of:

- One **solid-colored wire**
- One **white wire with a matching color stripe**

Example pairs:

- White/Orange + Orange
- White/Green + Green
- White/Blue + Blue
- White/Brown + Brown

These pairs must be placed in a specific order inside the RJ45 connector.

Incorrect ordering can prevent the cable from functioning properly.

## Why Wire Order Matters

---

Wire order determines:

- Signal integrity
- Pair matching
- Cable type (straight-through or crossover)

Incorrect ordering can cause:

- Link failures
- Packet errors
- Reduced speeds
