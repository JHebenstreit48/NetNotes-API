# Static Routing: Manually Building the Routing Table

<hr class="dividerSection" />

### What Is Static Routing?

<hr class="dividerSection" />

Routing is the process of moving packets between networks.

With <span class="emphasis">static routing</span>, routes are manually entered by the administrator.

This approach is simple but not scalable.

<hr class="dividerSection" />

### Static Routing Advantages

<hr class="dividerSection" />

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>Simple and predictable</li>
    <li>No protocol overhead</li>
    <li>Secure (no route advertisements)</li>
  </ul>
</div>

<hr class="dividerSection" />

### When Static Routing Is Used

<hr class="dividerSection" />

Most networks use a combination of static and dynamic routing, static for default gateways and key routes, dynamic for everything else.
