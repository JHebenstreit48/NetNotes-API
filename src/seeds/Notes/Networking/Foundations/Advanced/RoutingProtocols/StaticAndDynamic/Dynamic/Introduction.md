# Dynamic Routing: Letting Protocols Learn the Path

<hr class="dividerSection" />

### What Is Dynamic Routing?

<hr class="dividerSection" />

Routing is the process of moving packets between networks.

With <span class="emphasis">dynamic routing</span>, routes are learned and updated automatically using routing protocols.

This approach scales well for large networks.

<hr class="dividerSection" />

### Dynamic Routing Advantages

<hr class="dividerSection" />

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>Adapts to network changes automatically</li>
    <li>Scales well</li>
    <li>Reduces manual errors</li>
  </ul>
</div>

<hr class="dividerSection" />

### When Dynamic Routing Is Used

<hr class="dividerSection" />

Most networks use a combination of static and dynamic routing, static for default gateways and key routes, dynamic for everything else.
