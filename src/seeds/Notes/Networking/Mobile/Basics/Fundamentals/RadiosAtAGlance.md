
# Introduction

---
### Overview of Mobile Radios
---

Smartphones contain multiple radios that enable different communication services.

Each wireless function uses a specific technology optimized for its purpose, range, and power requirements.

### How Devices Choose a Radio
---

Mobile devices automatically switch between radios depending on:

- Signal strength  
- Availability  
- Power consumption  
- Configuration settings  

Each technology is optimized for a different communication model:

- Long-range communication → Cellular  
- Local networking → Wi-Fi  
- Short-range pairing → Bluetooth  
- Very short-range secure exchange → NFC  
- Global positioning → GPS  

---
### Relationship to Wireless Spectrum
---

Wi-Fi and Bluetooth operate primarily in the <span class="emphasis">unlicensed 2.4 GHz and 5 GHz frequency bands</span>.

These bands are shared by many devices, which can lead to congestion and interference in dense environments.
