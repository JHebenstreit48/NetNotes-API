## Radios at a Glance
---

Smartphones contain multiple radios that enable different services.  
Each wireless function uses a specific technology optimized for that purpose.

Mobile devices automatically switch between radios depending on signal strength, availability, and configuration.

Each technology is optimized for:

- Long-range communication → Cellular
- Local networking → Wi-Fi
- Short-range pairing → Bluetooth
- Very short-range secure exchange → NFC
- Global positioning → GPS
