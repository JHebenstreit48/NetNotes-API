# What Is POP3?

<hr class="dividerSection" />

<span class="emphasis">POP3</span> stands for <span class="emphasis">Post Office Protocol version 3</span>. A server that supports POP3 clients receives and stores messages addressed to its users.

<div class="centeredBullet">
  <ul class="diamondBullets fullWidthBullet">
    <li>When the client connects to the email server, messages are downloaded to the client.</li>
    <li>By default, messages are not kept on the server after they have been accessed by the client.</li>
    <li>Clients contact POP3 servers on <span class="emphasis">TCP port 110</span>.</li>
  </ul>
</div>

POP3 is now widely considered a legacy protocol, most major providers still support it, but recommend IMAP instead, since IMAP keeps mail synchronized across multiple devices rather than downloading it to just one.
