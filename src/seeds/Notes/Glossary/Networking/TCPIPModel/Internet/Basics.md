# Glossary - Internet - Basics

<hr class="dividerSection" />

### IP Address

<hr class="dividerSection" />

Logical identifier assigned to devices on a network to enable communication (Layer 3 of the TCP/IP model).

### Broadcast Domain

<hr class="dividerSection" />

Logical network segment where all devices receive broadcast frames sent by any device in that domain.

Switches flood broadcasts; routers do not forward them.

### Default Gateway

<hr class="dividerSection" />

The IP address of the router interface that hosts use to reach remote networks or the Internet.

### Regional Internet Registry (RIR)

<hr class="dividerSection" />

One of five organizations that allocate public IPv4 and IPv6 address blocks from IANA to ISPs and organizations.

The five RIRs are ARIN, RIPE NCC, APNIC, AfriNIC, and LACNIC.

<hr class="dividerSection" />

### DHCP Discover Packet

<hr class="dividerSection" />

Broadcast message sent by a client to locate any available DHCP server on the network.

Includes the client's MAC address.

### DHCP Offer Packet

<hr class="dividerSection" />

Response from a DHCP server to the client's Discover message, offering an available IP address, subnet mask, default gateway, and other configuration details.

### DHCP Request Packet

<hr class="dividerSection" />

Message sent by the client to accept the offered IP address from the DHCP server.

### DHCP Acknowledge (ACK) Packet

<hr class="dividerSection" />

Final confirmation from the DHCP server to the client, assigning the IP address and updating its table with the client's MAC address.
